- Το πρόγραμμα να μεταφερθεί σε C++.

- Να αναπτυχθεί γραφικό περιβάλλον CAD.
	- Αυτό θα μπορούσε να γίνει ακόμα και σε web περιβάλλον με WebGL αλλά χρειάζεται server (έστω τοπικός) για να στέλνονται τα δεδομένα στο solver.

- Ανάλυση
	- Να υποστηρίζεται πλαστική ανάλυση push over.
	- Να υποστηρίζεται δυναμική ανάλυση φάσματος.
	- Να υποστηρίζεται δυναμική ανάλυση χρονοσειράς.
	- Να υποστηρίζονται μεγάλες παραμορφώσεις.
		- Σώματα που πρέπει να παραμείνουν στερεά, δεν παραμένουν. Μετά από κάθε κύκλο τα κάνουμε στερεά, μετακινώντας τους αντίστοιχους κόμβους και αυξάνοντας τα αντίστοιχα επικόμβια φορτία.
	- Να υποστηρίζονται μη γραμμικά στοιχεία, π.χ. καλώδια και ρωγμές.

Υλικά
	- Να υποστηρίζονται ορθότροπα υλικά.
	- Να υποστηρίζονται σύμμικτα υλικά.
	- Να υποστηρίζεται μη γραμμικό συνάρτηση E στη σχέση σ = E(ε)
	- Μη γραμμικά στοιχεία υλικού όπως το fy.
	- Πυκνότητα.

- Διατομές Πλαισίου
	- Να υποστηρίζεται μεταβλητή διατομή.
		- <linear> με 2 ίδιου τύπου elements μέσα.
	- Να φτιαχτεί με συνοριακά στοιχεία (ακόμα και σαν ανεξάρτητη εφαρμογή) solver για επίλυση σταθερών τυχαίας διατομής πλαισίου.

- Φορτία
	- Να υποστηρίζονται ίδια βάρη
	- Να υποστηρίζονται συνδυασμοί φορτίων
	- Να υποστηρίζονται κινητά φορτία
	- Να υποστηρίζονται κατανεμημένα φορτία σε επιφάνειες και όγκους
	- EC1 για χιόνια, άνεμο (ακόμα και σαν ανεξάρτητη εφαρμογή)

- Κόμβοι
	- Απεριόριστοι βαθμοί ελευθερίας (όχι μόνο 6)
	- Να υποστηρίζονται βαθμοί ελευθερίας σε πακέτα διανυσμάτων 2 ή 3 διαστάσεων (π.χ. ux, uy, uz και θx, θy, θz)

- Στοιχεία
	- Η ράβδος να ενσωματωθεί στη δοκό.
	- Κάθε στοιχείο να μπορεί να έχει ελεύθερους επιλεγμένους βαθμούς ελευθερίας σε κόμβους του (π.χ. η δοκός να γίνεται ράβδος).
		- Αυτό επιτυγχάνεται με συμπύκνωση
	- Στοιχείο πλακών, κελυφών, όγκου, επιφανειακά
	- Μη γραμμικά στοιχεία (καλώδια, μεμβράνες)
	- Έδαφος

- Μετά Ανάλυση Επεξεργασία
	- Τάσεις, παραμορφώσεις, μετακινήσεις, εντατικά μεγέθη.
	- Συνάρτηση σχήματος με κατανεμημένα φορτία.
	- Επάρκεια κατά EC
		- Χάλυβας
		- Σκυρόδεμα
		- Αλουμίνιο
		- Ξυλεία
		- Τοιχοποιία
	- Βελτιστοποίηση διατομής
		- Συνεχείς αναλύσεις.
